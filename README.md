# Infrastructure as Code : NGINX sur AWS avec Terraform
Scénario du Laboratoire 1: Déploiement automatisé d'un serveur web NGINX sur AWS avec Terraform
## Contexte :
Une entreprise demande de déployer un site web dans leur compte AWS. Ils souhaitent que le serveur soit hébergé sur une instance EC2. Ma mission est d'automatiser le déploiement de cette infrastructure en utilisant Terraform.
